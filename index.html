DICE-GAME

Welcome to DICE-GAME, a fun and interactive game of luck and chance! Test your luck by rolling two dice and see who emerges as the winner.

DICE-GAME is a simple web-based game created using HTML, CSS, and JavaScript. It allows two players to take turns rolling dice, and the player with the higher roll wins the game. The game displays the winner's name or declares a draw if the rolls are the same.

Drum-Kit Webpage

The drum kit webpage is a dynamic musical interface crafted with HTML, CSS, and JavaScript. Users can simulate playing drums by clicking or using keyboard keys corresponding to different drum sounds. The interactive design combines vibrant visuals with responsive animations, providing an engaging and immersive experience.

The JavaScript code handles user input and triggers the appropriate drum sounds, creating a seamless interaction. This project showcases the integration of front-end technologies to produce an entertaining virtual drumming experience that's accessible and enjoyable for users of all skill levels.
